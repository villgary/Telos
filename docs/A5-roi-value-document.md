### 2.3 未被发现账号泄露的隐性成本 / Hidden Costs of Undiscovered Account Breaches

除了已发生的泄露，更大的风险是**未被发现的长期泄露**：

| 隐性成本 / Hidden Cost | 说明 / Description |
|---------------------|-------------------|
| **驻留时间成本 / Dwell time cost** | 攻击者在网络中平均驻留 207 天，每天造成数据外泄损失 |
| **横向移动积累 / Lateral movement accumulation** | 每增加一跳，损失呈指数增长 |
| **数据完整性损失 / Data integrity loss** | 被篡改的数据/系统难以评估，隐性重建成本极高 |
| **合规违规罚款 / Compliance fines** | 驻留时间内 GDPR/CCPA 违规累计 |

**关键洞察 / Key Insight:**
> MTTD（平均发现时间）207 天，每多驻留一天，平均损失约 $21,000 美元（约 ¥15 万人民币）。**将 MTTD 从 207 天缩短至 24 小时，可节省潜在损失约 $385 万美元。**

```
年化总价值 = 威胁发现节省 + 响应效率节省 + 合规成本节省 + 主动风险消除价值

威胁发现节省 = (基准 MTTD - Telos MTTD) × 每日驻留损失 × 预期攻击频率

合规成本节省 = 年度人工节省 + 避免的合规罚款
主动风险消除 = 年化泄露概率 × 降低比例 × 平均泄露成本
```
